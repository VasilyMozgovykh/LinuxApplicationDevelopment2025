# Д/З 12. Финальный проект
Взял за основу предыдущее Д/З про документацию игры-угадайки и дополнил необходимым функционалом (юнит-тесты, динамические библиотеки, инсталляция и т.д.)

## Сборка

Настроены `autotools` с выгонкой документации и man page
```sh
autoreconf -fisv;
./configure --prefix=<PREFIX>;
make install;
```
Здесь `<PREFIX>` - это папка, в которую устанавливаем программу. При инсталляции от `root` это может быть `/usr`.

## Опции программы
Опции программы на двух языках представлены в `help`-строке, которую можно получить при запуске
```sh
<PREFIX>/bin/guess --help
```
Также есть флаги
- `--version` - показывает версию программы
- `-r` - использует римские цифры вместо арабских

## Запуск юнит-тестов

В проекте всего 2 библиотечные функции для перевода из арабских цифр в римские, для каждой из них по одному тест-кейсу, оба в одном тест-сьюте
```sh
make check;
```

## Документация и manpage
Для сборки документации можно вызвать
```sh
make docs;
```
Страница `man` будет доступна в `doxygen-doc/man/man1/guess.1`

## Очистка
```sh
make uninstall;
make gitclean;
```
